# GLISS — Static Site

Pixel-for-pixel clone of the GLISS V2 site. Pure HTML, CSS, and JavaScript. No build step, no dependencies, no frameworks.

## Deploy to GitHub Pages

1. Create a new repository on GitHub.
2. Upload every file to the repo root.
3. Create an `images/` folder and upload your images (see list below).
4. Go to **Settings → Pages**.
5. Set **Source** to `Deploy from a branch`.
6. Select branch `main` and folder `/ (root)`.
7. Click **Save** — your site goes live at `https://yourusername.github.io/reponame/`.

## Customizing

**WhatsApp number:** Search for `27714636308` across all `.html` files and `main.js` and replace with your number.

**Contact details:** In `contact.html`, update the email, phone, and location in the contact info section.
